# دليل Windows الكامل - Windows Complete Guide

## 🎯 هذا هو أبسط شيء ممكن!

أنت تملك **3 ملفات فقط**، انقر عليهم بالترتيب:

### 1️⃣ setup_windows.bat
- **ماذا يفعل:** تثبيت كل المكتبات وإنشاء قاعدة البيانات
- **متى:** أول مرة فقط
- **كيف:** انقر مرتين على الملف
- **الانتظار:** حوالي 5-10 دقائق

```
سيشوف:
✅ تم إعداد كل شيء بنجاح!
```

### 2️⃣ start_backend.bat
- **ماذا يفعل:** شغّل الخادم (Backend)
- **متى:** قبل start_frontend.bat
- **كيف:** انقر مرتين على الملف
- **الانتظار:** حتى تشوف `Uvicorn running on http://127.0.0.1:8000`

```
لا تغلق هذه النافذة!
```

### 3️⃣ start_frontend.bat
- **ماذا يفعل:** شغّل التطبيق (Frontend)
- **متى:** بعد شغلك start_backend.bat
- **كيف:** انقر مرتين على الملف في **نافذة منفصلة**
- **النتيجة:** المتصفح يفتح تلقائياً على http://localhost:3000

```
لا تغلق هذه النافذة!
```

---

## 🚀 الخطوات الفعلية:

1. **افتح مجلد المشروع** في Windows Explorer
2. **انقر مرتين على** `setup_windows.bat` ← **شغّل مرة واحدة فقط**
3. **انتظر حتى ينتهي** (قد يستغرق 10 دقائق)
4. **انقر مرتين على** `start_backend.bat` ← **شغّل دائماً أولاً**
5. **في نافذة جديدة، انقر مرتين على** `start_frontend.bat` ← **شغّل ثانياً**
6. **المتصفح سيفتح تلقائياً!** ✨

---

## ✅ كيف تعرف أن كل شيء يعمل؟

- ✅ نافذة Backend تقول: `Uvicorn running on http://127.0.0.1:8000`
- ✅ نافذة Frontend تقول: `Compiled successfully!`
- ✅ المتصفح على: `http://localhost:3000`
- ✅ تطبيق جميل يظهر بـ واجهة عربية

## ❌ إذا حدثت مشكلة:

### مشكلة: "Python not found"
```
✅ الحل: ثب��ت Python من https://www.python.org/downloads/
```

### مشكلة: "Node is not recognized"
```
✅ الحل: ثبّت Node.js من https://nodejs.org/
```

### مشكلة: "firebase-key.json not found"
```
✅ الحل: تأكد من أن الملف في: backend/firebase-key.json
```

### مشكلة: setup_windows.bat لا يعمل
```
✅ الحل: 
1. اضغط Windows + R
2. اكتب: cmd
3. اضغط Enter
4. اكتب: cd [مسار المشروع]
5. اكتب: setup_windows.bat
```
