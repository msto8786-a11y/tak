Industrial Control Lab Simulator - Plesk Deployment Guide

📦 محتوى هذا المجلد:
================================================================
- index.html              ← صفحة الموقع الرئيسية
- static/                 ← ملفات JavaScript و CSS و الصور
- asset-manifest.json     ← فهرس الملفات
- README.txt              ← هذا الملف

================================================================
📤 خطوات الرفع على Plesk:
================================================================

١. سجّل دخول إلى Plesk → Domains → اختر الدومين الخاص بك.

٢. اضغط "File Manager".

٣. ادخل إلى مجلد "httpdocs" (وفي بعض الإصدارات اسمه "public_html").

٤. احذف أي ملفات قديمة موجودة (index.html القديم مثلاً).

٥. اضغط Upload وارفع كل محتويات هذا المجلد:
   - index.html
   - asset-manifest.json
   - مجلد static/ (كاملاً)

   ✅ ملاحظة: يمكنك ضغط هذا المجلد كـ ZIP ورفعه ثم استخراجه عبر File Manager.

٦. تأكد أن SSL مفعّل: SSL/TLS Certificates → Let's Encrypt (مجاني).

٧. افتح الدومين في المتصفح - يجب أن يظهر المحاكي.

================================================================
⚡ ربط وحدة ESP8266 مع الريليه:
================================================================

⚠️ مهم جداً: بسبب قواعد المتصفحات الأمنية (Mixed Content)،
المحاكي على HTTPS لا يستطيع الاتصال بـ ESP8266 على HTTP مباشرة.

🟢 الحل (لتشغيل المحاكي مع الـ ESP8266):

الطريقة ١ - الأفضل لمشروع التخرج:
- شغّل المحاكي على شبكة الجامعة المحلية على HTTP فقط (بدون SSL).
- اضبط ESP8266 على نفس شبكة Wi-Fi.
- افتح المحاكي بـ http://localhost أو http://192.168.x.x.
- ادخل IP الـ ESP8266 في إعدادات الواي فاي.

الطريقة ٢ - للعرض البسيط:
- استخدم Chrome مع تعطيل أمان المحتوى المختلط:
  Settings → Privacy → Site Settings → Insecure content → Allow

================================================================
💾 حول حفظ الجلسات:
================================================================
- الجلسات تُحفظ تلقائياً في متصفح الطالب (LocalStorage).
- كل طالب يحفظ في متصفحه الخاص (لا تختلط البيانات).
- يمكن للطالب تصدير جلساته كملف JSON ومشاركته مع المدرس.
- المدرس يستطيع استيراد ملف الطالب لمراجعة عمله.
